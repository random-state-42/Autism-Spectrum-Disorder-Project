About the Project

This Autism Spectrum Disorder Detection System is designed to assist in the early identification of autism traits using a simple questionnaire-based method. The goal is to provide a preliminary risk assessment that can help guide further evaluation and professional consultation.

Key Features

  • 10-question survey tailored to detect common behavioral indicators
  • Machine learning model classifies responses into risk categories (low, moderate, high)
  • Displays precautionary advice and suggestions based on risk level
  • User-friendly and mobile-responsive interface

Technology Stack

  • Frontend: HTML, CSS (custom styling)
  • Backend: Python with Flask framework
  • Machine Learning: Trained classifier model (XGBoost classifier)
  • Deployment: Local server
